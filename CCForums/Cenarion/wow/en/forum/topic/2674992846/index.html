For those that don't know, before I quit I concentrated on soloing current content - 5 mans and then 5 man heroics. I then went back to solo older 25 and 40 man raids. While nothing special, some older content is harder to solo than current content, and some events are still unsoloable.

So since my pet takes a massive amount of damage, which I try my best to heal through, and I'm able to keep myself alive as best as I can, I think hunters should be able to queue as a healer.

There are a few minor tweaks that shouldn't affect anything else other than healing. I think most hunters would agree with me and those who don't are probably @%%*'s anyways, so who cares what they think?

1. Allow spirit mend (that's the spirit beast ability I think) to be baseline for all hunters and not have a CD. Blizzard not already doing this shows that probably none of them play a hunter.

2. Change the 31 point talent to allow the hunters to use the current pet special onto any party member, including the hunter. This would allow the hunter to throw a shell on the tank or allow the tank to cower. It would also allow the hunter to force the mage to use mangle. Not changing this is a slap in the face.

3. Allow the move to command to be usable on party members so when the tank doesn't move out of the green stuff the hunter can force them to move. Likewise, the hunter should be able to use the passive command to force the target party member to stop doing whatever and run to the hunter. This should fix hunter pvp, but blizzard obviously hates hunters.

4. Give the hunter a rez. Rename rez pet to "Rez whatever you want". Offer a glyph for it so it's instant. This probably won't happen cuz blizzard won't let us ride our pets so they probably won't let us do this which is exactly what a hunter is.

5. Allow hunters to use feign death for every party member, so if a pull goes bad, everyone falls down, the mobs get confused, and go back home and watch tv or whatever they do when not fighting. This should have been done in alpha so blizzard needs to get with the times.

Please thumbs up, favorite, request sticky, and whatever else you can do to get this to the developers attention.

As am I. I'm definitely onboard with this idea. But honestly, I think SV makes more sense as a healing spec. I mean come on, it's called "survival." Here's some ideas that I have on this.

• Change widow venom (with a talent in the SV tree) to help friendly targets. You could still use it as as the reduced healing debuff it is now. But if you hit a friendly target with it, it provides a healing buff instead. To balance it out though, you could only have one widow venom ticking at any one time (i.e. a friendly target or an enemy, not both).

• Give us a new aspect that increases healing. I can't think of a name for this one though. But it would basically give a +75% bonus to all healing done. I know that sounds high, but it needs to be in order to cover the fact that we don't have enough heals already.

• An aspect that lowers damage taken to all party members (aspect of the chicken hawk). This one would be used if you were doing more party healing than tank healing, obviously. You could macro the two aspects to swap for different healing phases as well.

These are just a few ideas. I could think up some others, but I don't want to hijack your thread. Anyway, great ideas. I too like to solo old content. So I think I have a little understanding on how Blizz should add this healing idea. I mean come on, people are constantly complaining about hunter healing. This would fix it completely.
