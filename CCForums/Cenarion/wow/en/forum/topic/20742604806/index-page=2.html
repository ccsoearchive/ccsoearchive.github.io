I'm confused as to why people actually thought they were removing conversations. Even if the wording is bad.


If you read the thread, it's obvious they are.

This is the only other blue response in that thread:

Hello, thank you all for your feedback so far. At this point in time I don't have any additional information regarding these changes. If I get any further details I will be sure to communicate them with you. In the meantime please feel free to continue leaving any additional thoughts or feedback that you have.


Considering she's avoiding denying the removal of conversations, it seems pretty obvious that is the current plan.

The post is in regards to addons, yes, but the APIs they mentioned, "BNCreateConversation" and such are necessary by the game to do conversations. Addon users that alter the chat log, (like WoWInstantMessenger, for instance), have to access the base commands in order to get the text. Thus, they mentioned them. Based on what they said, they're completely removing the capability of the game to create or join a conversation via Battle.net, which is necessary for cross-faction chatting or chatting with friends on different servers at the same time.

So until Blizzard says otherwise (which they haven't yet), the official word is indeed, that conversations are being removed. Just because the post was towards addon authors doesn't mean anything. They're removing support for conversations, which means conversations won't exist.

I don't think removing computer programming function with "Conversation" in the name means they are removing battle.net chat. They are changing the API. Older functions will be replaced by new ones.

But it does look like they're removing the group chat without adding in a replacement feature in the near future. That's going to leave the players who were using it scrambling for other options.

It's going to be a hard adjustment for the RP community since they're used to conversing in text. You don't always want to have everyone in the same group during an event to use group/raid chat and it's impossible if it's a cross faction event like the OP was talking about. Having to constantly tab outside of WoW to a different program that handles group chat isn't a good alternative, especially for people who run the game full screen, and using voice comm would present some interesting complications for RP.

Some of the long time role players are going to have to remember how they handled things before Battle.net was added.
