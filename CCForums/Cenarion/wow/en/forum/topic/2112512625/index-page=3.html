Inside the data folder create the following folder:

Sounds

Inside the Sounds folder you just made, create the following folder:

Spells

Your folder structure should now look like this:

C:\Program Files\World of Warcraft\Data\Sounds\Spells

Inside the Spells folder is where you put your two blank .wav files, one named AspectofTheHawk.wav the other named Pathfinding.wav

XP/7/Vista doesn't matter, just get the Spells and Sounds folders created inside the Data folder in your Warcraft install.

Hope this I cleared the water a little and didn't muddy it :) Good luck!

Macro Update:

#showtooltip SHOT NAME
/startattack
/castsequence reset=1 !ASPECT NAME, null
/cast SHOT NAME

I've found this to be much smoother, I can't take credit, I think Cupz posted them on AJ.


Concerning XP putting on .wav automatically, you probably have the option to View Extensions for Known File Types toggled. Open My Computer, hit View, Folder Options, and I think it's in the middle of the page. Windows won't let you change file extensions unless you have this option toggled.
