I don't know if a similar macro has been posted here, but I modified this macro I found at howtopriest.com to do the following:

-Automatically casts whatever talent you have chosen
-Automatically changes the icon (on first cast after you switch)
-Automatically changes the tooltip for that button (on first cast after you switch)[/li]

*credit to Spinalcrack, whose macro I modified. This macro has been tested and works as expected. I have a level 60 version of this as well.

*****NOTE THAT YOU HAVE TO NAME YOUR MACRO "90 Talents" or change the code in the macro to reference your name for this macro to work.

#showtooltip
/cast Glaive Toss
/cast Powershot
/cast Barrage
/script gt=GetSpellInfo(117050); ps=GetSpellInfo(109259); b=GetSpellInfo(120360)
/run SetMacroSpell("90 Talents", GetSpellInfo(gt) or GetSpellInfo(ps) or GetSpellInfo(b))
