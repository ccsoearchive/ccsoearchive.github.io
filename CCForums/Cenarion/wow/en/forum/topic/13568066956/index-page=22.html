Zarhym,
Can you provide any details on the Area-52 armory issue?
Just some information on what is affecting things.

Keep the techies satisfied for a while!

After an initial investigation it looks like character profiles for Area 52 are being processed and saved into an outdated folder location following connected realm maintenance on 7/24, so the web Armory is looking for them in the wrong folder in our database. That's why Area 52 players are receiving "Character Not Available" errors rather than simply seeing outdated info. I don't know how fast/easy this is to fix, but the good news is that profiles should properly be updating once the folder location is updated, as the game is properly reporting character info in our database for the web Armory to pull from.
