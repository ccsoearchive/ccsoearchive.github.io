01/06/2011 2:44 AMPosted by Zate
so for OH i needa know, why throat slasher > vicious glad shiv.. all u do is gain hit, just reforge ur crit into hit wala, hit, plus more DPS, and top end
EP values are: DPS (1.04), agility (2.6), hit (1.4), crit (0.9), resilience (0.0). Reforging for mastery (1.3).

Throat Slasher: 409.6(1.04) + 129(2.6) + 86(1.4) + 52(0.9) + 34(1.3) = 972.784.
Vicious Gladiator's Shiv: 462.3(1.04) + 146(2.6) + 59(0.9) + 38(1.4) + 97(0.0) = 966.692.
Vicious Gladiator's Shiv: 462.3(1.04) + 146(2.6) + 59(0.9) + 38(1.3) + 97(0.0) = 962.892.

First shiv reforged for spell hit, second for mastery. The throat slasher could only be reforged for mastery since it already had hit, etc. So they're very close, but no cigar just due to the wasted resilience itemization (which cannot be reforged).

I'll be revisiting all the listed EP values today, so if these are off compared to the earlier post they'll be corrected by this evening. The ones in this post are correct though. DPS/damage isn't as significant for the OH as the MH, so other values tend to be more important.
