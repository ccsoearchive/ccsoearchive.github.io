I'm not sure if this has been pointed out or not yet, but if you find yourself low on mana using SW:D on the target will get you 10% of your overall mana back. This comes from the talent "Masochism." Another thing I want to point out is that (unless they've changed this) SW:P keeps all SP boosts you have at the time it is first applied. So for longer fights I have actually gotten higher numbers if I use the following opening rotation. VT -> MB -> DP -> MFx2 ->SW:P. I do this because by speccing into Evangelism we get back our older talent "Shadow Weaving." So by having Evangelism up to 5 stacks before applying SW:P we get an extra 10% damage to SW:P.
