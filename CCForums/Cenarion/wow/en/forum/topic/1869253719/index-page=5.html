Diminished dodge/parry from undiminished dodge/parry ( y = diminished, x = undiminished,
k = class constant, c = dodge/parry cap):

y = (kxc)/(kx + c)

Note x = rating/176.719

Clearly this is non-linear

Let x approach infinity:

Y = lim (x -> inf) [c/(1+c/kx)]

Y = c

This means you need an infinite amount of dodge/parry rating to hit the dodge/parry cap (c)

Take Derivative of y with respect to x:

dy/dx = (kc^2)/[k^2*x^2+2kcx+c^2]

Note the larger x becomes the smaller dy/dx is. This means the more dodge/parry rating you get the slower your diminshed rating grows.

Saw alot of arguments in this thread about 'linear curves' and 'diminishing returns doesn't become more apparent as you get more dodge/parry rating'. My math shows this to be untrue.

It takes:
176.71899 dodge/parry rating to get 1% before Diminishing Returns
179.28 mastery rating to get 1 Mastery.

1 Mastery gives you 2.25% chance to block.
As long as you keep holy shield up, you block 40% of incoming damage.
2.25% chance to block 40% of the incoming damage is 0.9% avoidance with no diminishing returns.

So all we really need to find out is:
When does diminishing returns on parry/dodge turn 179.28 rating into 0.9% avoidance.
In other words, when is 1.014491990928649% diminished to 0.9%

The formula for Diminishing Returns on Dodge/Parry for Paladins at 85 is:
1/x' = 1/65.631440 + .9560/x

Where x is the expected value of avoidance before diminishing returns and x' is the value you actually get after DR.
Since we already have values that we want to test for, we can create a second equation:
x/x' = 1.014491990928649/0.9

Now we have two formulas, each with two variables, and we can solve for the values of x and x'.

Turns out, x = 11.23697695234907 which makes x' = 9.96881133369682

Someone correct me if I'm wrong, but I believe that the character sheet shows your actual dodge/parry values AFTER diminishing returns.
If this is true, then it means once you hit 10% dodge/parry on your character sheet, you get more avoidance from Mastery from that point on.

This probably doesnt fit the topic and You probably dont have room for it anyways but I thought i would post the new cata formula for determining armor Damage Reduction.

The EH benefit of armor against physical damage does scale linearly but the Damage reduction aspect is affected by Diminishing Returns.

The Damage Reduction formula for armor is as follows:


Reduction = Armor / (Armor + MobLevel * 2167.5 - 158167.5)

http://blue.mmo-champion.com/topic/22463/physical-mitigation-change-intentional
