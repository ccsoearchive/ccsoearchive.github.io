The part in bold is wrong. And it's not the first time I read this on these forums, so here's the long math-ish story about it.

You're making the assumption that dodge and parry work that way because that's how armor works, but this is only true because the mathematical upper limit of the armor function (when [armor = ∞]) is 100%.

I'll save you the math. Let's just use simple examples.

With armor, if you have this:
[armor = x], [time-to-live = y]
... then you can say that:
[armor = 2x] <==> [time-to-live = 2y]
[armor = 3x] <==> [time-to-live = 3y]
[armor = 5x] <==> [time-to-live = 5y]
[armor = 30x] <==> [time-to-live = 30y]
[armor = 1,000x] <==> [time-to-live = 1,000y]
...
[armor = ∞] <==> [time-to-live = ∞]

In other words, if [armor = ∞], then [damage reduction = 100%]. (Ignore the hard cap at 75% damage reduction, which is separated from the armor function. Dodge, miss and parry have no hard cap.) Time to live is proportional to the value of the stat if (and only if) [time-to-live = ∞] when the value of the [stat = ∞]. If either value isn't ∞ when the other is, then the relation is not proportional.

I don't know the new values, but if I remember correctly, in Wrath of the Lich King, dodge had a limit at around 60% and parry at 40%, and miss at around 25%. I might be totally off with the numbers, but the point is that none of the avoidance function has a limit at 100%.

This means that, at [dodge rating = ∞], [damage reduction ≈ 60%], so [time to live ≠ ∞], so the concept of time-to-live that applies to armor does not apply to mitigation. If you like it better:
[dodge = x] <==> [time-to-live = y]
[dodge = 2x] <==> [time-to-live = ?y]
[dodge = 3x] <==> [time-to-live = ?y]
[dodge = 5x] <==> [time-to-live = ?y]
[dodge = 30x] <==> [time-to-live = ?y]
[dodge = 1,000x] <==> [time-to-live = ?y]
...
[dodge = ∞] <==> [time-to-live ≈ 60%]

Therefore, unless the new mathematical upper limit of the dodge and parry functions are at (or above) 100%, the value of those stats do diminish as their value increases, and after a while, it not really worth putting points into versus mastery which, in the case of a warrior, has augmenting returns.

Not that I really agree that armor doesn't have diminishing returns by the way, the longer is your time-to-life, the less effective are each additional +1 second of time to live relative to the previous one, versus +1% avoidance. Kinda like how you don't care about getting +10 health when you have 150,000 health at level 85, while you do when you have 400 health at level 30. But that's another thing.



Edit: TL;DR version:
Armor diminishing returns are not the same as parry and dodge (and miss if it still exists) diminishing returns. While some argue that armor has no diminishing returns because each point adds as much time-to-live as the previous one, this can't be said about parry and dodge.
