Not to steal Jo's thunder here, but I'll answer your question too Ruinei.

I remember hearing early on in beta (second wave of invites iirc) that Envenom was benefiting from the Pandemic affect but I never actually tested it, until now.

So, Envenom lasts 1+CPs Seconds and I'm going to be using 5 CPs so 6s. Pandemic affect allows a maximum of 30% of the duration to be added to the new application, so 6+(.3*6)=7.8s which will probably get rounded up to 8s on buff times.

When Doing Envenom->MfD->Envenom my duration goes to: 8 Seconds, which instantly goes down to 7 Seconds on the buff timer (probably due to it getting rounded up to 8s at 7.8s then back down once it goes down a little bit farther) . Which means it is (probably) getting Pandemic Affect.

The difference between my answer and Jo's is that he's using 7 seconds as the base duration (7+(.3*7)=9.1) instead of the actual base duration which is 6s at max CPs (which can be gotten from the tooltip)

On CT: the assassination APL for AoE is super hackish and kinda hard to read.
The line
actions+=/rupture,cycle_targets=1,if=active_enemies>1&!ticking&combo_points=5
basically says use rupture as a finisher if there is a target that doesn't currently have rupture ticking put rupture on that target. This line is before the CT line so if there is a target without rupture on him that will be used instead of CT. CT is used as the filler finisher if all targets have rupture on them.

AoE rotations also add all sorts of tricky caveats about "if target will live full duration" which complicate a lot of advice about AoE. I'm still trying to figure out how to talk about AoE for the Ravenholdt guide because its really hard to generalize.
