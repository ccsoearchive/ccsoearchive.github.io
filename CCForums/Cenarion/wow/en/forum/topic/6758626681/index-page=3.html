Great guide. So was I seeing it correctly that haste was our top stat as combat? So like over pure agi gems in red sockets use an agi/haste one?


I posted this in another thread, but this should answer your question.

Keep in mind that gemming pure haste in a red socket is only viable if the set bonus for gemming a red does not outweigh it.

Assassination:

Agility stat weight: 2.729

Mastery stat weight: 1.337

Delicate Primordial Ruby - 160 agi - Total Value = 436.64

160 * 2.729 = 436.64

Adept Vermillion Onyx - 80 agi and 160 mastery - Total Value = 432.24

80 * 2.729 = 218.32

160 * 1.337 = 213.92

218.32 + 213.92 = 432.24

Combat:

Agility stat weight: 2.770

Haste stat weight: 1.635

Delicate Primordial Ruby - 160 agi - Total Weight = 443.2

160 * 2.770 = 443.2

Deft Vermillion Onyx - 80 agi and 160 haste - Total Value = 483.2

80 * 2.770 = 221.6

160 * 1.635 = 261.6

221.6 + 261.6 = 483.2

Quick Sun's Radiance - 320 Haste - Total Value = 523.2

320 * 1.635 = 523.2

The enchants are listed so that the best one is on top, and the ones below it decrease in value as you go.

So for:

Weapons: Enchant Weapon - Dancing Steel.
Enchant Weapon - Windsong.
Enchant Weapon - Elemental Force.


Dancing Steel > Windsong > Elemental Force

Also worth mentioning on the Windsong enchant, since you mentioned the crit proc:
Unless it has changed, it's possible to have all three (crit, haste, mastery) procs active at the same time. So, although it is random, having the crit buff up wouldn't prevent you from getting your wanted mastery proc.
