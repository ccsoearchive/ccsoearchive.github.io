Few updates for the assassination side of things.

pulp:
for what it's worth... T14 Normal geared simcraft with different opening methods 25k iterations

  • mut -> snd -> mut(no dispatch here even if it procced) -> rupture - 87060
  • mut -> rupture -> mut (no dispatch) -> snd - 86983
  • mut -> rupture -> dispatch or mut -> snd - 86952
  • mut -> snd -> dispatch or mut -> rupture - 86929
  • amb -> snd -> mut -> rupture - 86909
  • garrote -> snd -> mut(s) + dispatch -> rupture (5cp) - 86859
  • amb -> rupture -> mut -> snd - 86796
  • mut -> mut -> dispatch or mut if not at 5cp -> rupture (5 cp) -> mut or dispatch if at 0 combo points -> snd - 86134

on each setup there was about a 25 dps margin of error and about a 6k -7k dps range when you threw out the top and bottom 5% of the results

So the implication yet again on openers is it just doesn't matter. mutilate vs ambush vs garrote and prioritizing snd vs rupture, the choice is yours. the only one i wouldn't recommend is the last. Putting off SnD to build a 5cp rupture first seems to hurt more than it helps.

http://elitistjerks.com/f78/t130892-assassination_mists/p6/#post2209599

Those are interesting numbers. Unfortunately I'm not really able to debate anything with you, this guide is basically a collection of the best peer-reviewed information I could find consolidated all in one place.

I'm familiar enough with most of it to roughly evaluate potential contradictions, but PvE subtlety is an exception - and since it's not a primary raiding spec in T14, there's not much research to read up on.

There are a couple things you could look into, though.

First is gear scaling; it's possible between N14 and H14 gear the order of stats change.
Second is dummy DPS is kinda... worthless. You need full raid buffs/debuffs to do any adequate tests.
Third... mastery should have no effect at all on ambush as far as I can tell. That's pretty weird.
Fourth, a 12K (~27%) DPS increase -just- from reforging is kind of... very, very crazy.

As in like, WAY practically impossible crazy.

I'm actually pretty sure that's not even mathematically possible within the constraints of the game.

So I'm not really sure what to make of it.

You're not the first PvE subtlety rogue I've seen say mastery is the best secondary stat for the spec after capping hit/expertise, I'm just not equipped to comment on it. It might make a good thread itself, though.

Edit:

To expand slightly on the math, this is what would have to happen to make those numbers work.

Crit would be worth 0.427 DPS per point.
Haste would be worth 1.459 DPS per point (3.4 times the value of crit).
Mastery would be worth 5.139 DPS per point (12 times the value of crit).

That's a pretty extraordinary difference to've gone unnoticed this long into a new expansion.

Especially since, for comparison, agility is only worth about 3.667 DPS per point.
