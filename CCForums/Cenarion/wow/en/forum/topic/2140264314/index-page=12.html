Took me a few months to get around to it, but I think I have something to contribute finally. ^_^

I have my default Blizzard UI set so that the only debuffs shown on the target frame are mine. I also have it set to display the health percentage all the time so I know when to switch to backstab.

http://img853.imageshack.us/img853/7180/addonsu.png

ComboPoints: Just so I have them somewhere in my face. Easy to place/resize, and you can change the color of each combo point individually using ingame slash commands.

eCastingBar: Shows what my target is casting, since the default cast bar is right on top of my focus icon the way I have things set up. I had it set to show for the screenshot, since target dummies never cast anything. Otherwise I have it set to be translucent unless there's an active cast (which also gets rid of the words under ComboPoints). There's also a bar that shows what your focus target is casting, which is handy.

Elkano's BuffBars: I LOVE THIS ADDON. I'm only using it to track rupture, slice and dice, envenom, and overkill, but you can do much more with it in terms of tracking buffs and debuffs on you/your target/pet/focus. I experimented with using it to track weapon enchant and trinket procs as well as the 4pc t11 proc, but it got overly cluttered and in the end it didn't change my playstyle at all anyway. It is an option though.

TellMeWhen: I LOVE THIS ADDON EVEN MORE. Here I'm just using it to track CDs really, but it's amazing on my shadow priest for tracking shadow orbs and dark evangelism right beside mind blast and archangel. You can also use it to track buffs, debuffs, and an almost ridiculous number of other things. You can set it up to prompt you based on preset conditions, although I haven't dug into it that heavily.

PS: Nicked everything but eCastingBar from Xerofour's UI: http://www.youtube.com/watch?v=7O9RibRejag

UI i use, been moving Need to Know a bit more to left to allow better vision of my character but i find this a pretty useful set up.

http://www.mediafire.com/imageview.php?quickkey=velqultn5i29q72&thumb=4

Action Bars are set up so that F1-12 are CDs, F1-5 being offensive.
1-7 Main Abilites/CC's.
Shift 1-7 Combo Abilites.
R, T, Y Utility's. U, I Mounts.


Add-ons
Bartender - Actionbar's
KgPanels - 3 Panels with the art along the bottom.
Omen - Threat meter.
Phanx Chat - Chat mod for better interactions and text command bar ontop of chat square.
Simple Mini Map - Moved and changed shape of Mini Map
Shadow Unit Frames - Unit Frames in lower right corner.
MiksScrollingBattleText - Scrolling Combat text to the right of my Character with "Static" ontop, static lists CD's that finish and Major buffs/Debuffs etc.
IceHud - Shows Players HP/Energy around Player, enemy's info on right side, also shows Combo Points above where i have my Need to Know.
Need to Know - Buff/Debuff tracking.
DBM - Boss mechanics mod in the lower left above panels.
Grid - Raid/Party status, use it for my heals, dont need it rogue but i like to know if people die.
Tidy Plates - Better names plates with dot/cc tracking ontop.

Seems overly much but i say knowledge is power, i use it all and love it. If i ever feel its to clutterly, i remove grid, move NTK further left and Scrolling Text further right.
