With the position I placed my Omen if I scaled it back, I'd just end up with a negative space border around it that wouldn't be giving me any extra situational awareness, or otherwise do me any good. That said, I positioned it in such a way that that's true because after playing around with it's scale I've found what is, for me, what feels like the right balance between real estate and ease of use. Larger print = easier on the eyes. At the size I have it I can very easily and quickly read relative threat positions without having to divert my main focus away from the action in the centre of my screen. This way I can see not only if I'm creeping up on the tank but other DPS, or worse yet, a healer and prepare if I need to use Tricks/Feint or chase down an errant mob before it gets to my healer. But yes, short story long, my threat meter is my precious. If I could have one add-on and one only I'd take a threat meter.

My UI:
http://img195.imageshack.us/img195/7413/wowscrnshot032411142424.jpg

Using Tukui, which I've modified to my liking. I don't use the Tukui raid frames, because I don't like them, and I've been to busy to learn how oUF works. I've been using the blizz raid frames instead.

Addons used:
DBM
Recount
Omen
AdiCCMonitor
InterruptNoSpam
Tukui_Filger (the icons showing my buffs/debuffs)
sCombo (dependent on Tukui)
NugEnergy (because last time I checked, sCombo didn't have a built-in way of displaying energy)
ComboPointsRedux (which I only use to track DP stacks, but it's really not necessary with filger showing it)
OnScreenHealth (only use it to show target HP%)
Tukui_Castbar (to separate the target castbar from the target frame)
TidyPlates
nibMicroMenu
Bagnon (Disabled the Tukui bag module, I prefer bagnon)
Align
Auctionator

I've used different addons for tracking rupture, snd, etc, but I've found just the icons with a timer to be the best for me.
