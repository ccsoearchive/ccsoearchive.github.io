Just eyeing up your gear, you've got plenty of unreforged pieces. If you've got a dodge/parry item, reforge whichever is higher to mastery, so on your chestpiece for example, reforge the dodge. If they're even, you want to reforge whichever you have more of. (They're pretty even for you at the moment, but in general, you'll wind up with more parry because of strength converting to parry and Blizzard liking putting parry on things). Also, upgrade your trinkets. If you can be bothered running dailies, the Tol Barad ones give Mirror of Broken Images, which is an amazing trinket. Another good one is Impetuous Query, which is a BoE from normal Lost City of Tol'Vir that you can usually find on the AH for a decent price.

Block DR formula:

This is what im getting from Thecks Sacred Duty blog on block:

Block chance=Base block+diminished block

For simplicity (and cause i cant use thecks symbols here) lets call Block chance B_c, Base block B_0, and Diminsihed block B_d.

Further B_cap will be the block cap and m will be mastery. k the avoidance constant we all know and love -_-

1/(B_d)=1/B_cap+k/m

B_cap=1.491 (not sure if this is at level 85 or not)
k=same as avoidance at 0.885
m=mastery rating (i think)

Now theck then finds how the block chance changes with respect to mastery:

d(B_c) = d(B_d)=-[(1/B_cap+k/m)^(-2)]*[-k/(m^2)]
d(B_c)=[(1/B_cap+k/m)^(-2)]*[k/(m^2)]=k*[(B_d/m)^2]

Avoidance formulas:

Base Parry=3%
Base Dodge=5.01%

Blood Elf Base Strength=176
Dodge cap=65.63144%
Parry cap=235.5%

Dodge = base dodge + (1/Dodge cap + k/pre dodge)^-1
Parry = base parry + Q/base Str+(1/parry cap + k/((total str - base str)/Q + pre-parry))^-1

Q=str to parry conversion=952

Armor formula:

M_ar = Ar / (Ar + K)

M_ar=mitigation from armor
Ar=armor
K=58370 at level 93
