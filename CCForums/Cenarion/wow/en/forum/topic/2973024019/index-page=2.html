Awesome post, OP. I'm still heavily of the mindset that if there is a balance that needs to be made for PVP, make that effect only function in PVP. The game needs to be divided the same way Guild Wars was, as that game's PVP fueled teamwork and balance (save for Warrior/Monks), even in pick-up groups, far more than WoW ever has.

For example, the Mesmer class from Guild Wars. It was a very fresh, new class that hadn't really been seen in an MMO yet. Warriors were tanks and heavy melee damage, while Rangers were "pullers" and would pick off enemies before they reached the party, and Monks healed and resurrected...

Mesmers, however, specialized in doing absolutely nothing but $@!&ing your day up. At first glance, I, at least, would have thought this kind of character would never be able to function in both aspects of the game. But if you read the skills, many of the Mesmer's say this:

Accumulated Pain
Spell. Deals 15...63...75 damage. Inflicts Deep Wound condition (5...17...20 seconds) if target foe has 2 or more hexes.

But when you entered a PVP match, the skill would change to this:

Accumulated Pain (PvP)
Spell. Deals 10...30...35 damage. Inflicts Deep Wound condition (5...17...20 seconds) if target foe has 2 or more hexes.

That's a basic example, with some other abilities changing almost entirely for the sake of PVP. Why this kind of thing was never done, I'll never understand.
