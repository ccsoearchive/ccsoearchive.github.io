I made a macro for running as an engineer, With the Nirto Boost tinker on there belt.


/use (your tinkered belt)
/cast !Sprint

-This alows you to have one Button/key to use to "run" rather than have two seperate ones to boost and sprint.
-First use is the nitro boost
-If the nitro boost is on CD then hitting this macro will cast sprint.
-you can edit this macro to were the icon shows the cooldown simply by adding
#showtooltip (Your Tinkered belt)

For Ex.

#showtooltip Ruthless Gladiator's Waistband of Accuracy
/use Ruthless Gladiator's Waistband of Accuracy
/cast !Sprint

Just make sure that the Tinkered belt you are wearing is typed into the first two lines of the macro.

You can get very creative with this macro and do something For compleatly runing away like

#showtooltip (your tinkered belt)
/use (your tinkered belt)
/cast !Sprint
/cast Cloak of Shadows
/cast Vanish
/cast Recuperate

But like i said "Just make sure that the Tinkered belt you are wearing is typed into the first two lines of the macro."

That's not technically a mod, and so long as you construct your macro in good order, you shouldn't need to use it ever under normal circumstances as one. Its main use is to instruct macros to create a fall through when using abilities that have modifiers assigned to them. That way the ability or item in question only activates with one of the macros main abilities, the one without modifier is used. Trinkets, Shadow Dance, Shadowstep are common uses of this for rogues. This way you can use the same key for other abilities, but not waste CDs at inopportune times.

Don't really know if it will be helpful, but ...

For sub guys as myself, this is quite helpful! Since now Shadow Dance and Stealth HAVE the same bar ...

#showtooltips
/cast [stance:1]Ambush;[stance:3]Ambush;[nostealth]Backstab
/cast [stance:1/3]Premeditation

It works like this:
- without stealth (nostealth), it will be Backstab;
- with stealth (stance:1), it will be Ambush;
- with Shadow Dance (stance:3), it will be Ambush;
- both on stealth and Shadow Dance, it will also automatically use Premeditation

You can change the (stance:1) ability if you want, well, since Ambush with stealth ( w/o Shadow Dance ) have 60 energy cost, it isn't profitable, so I'd recommend Crimson Tempest or Backstab itself.
