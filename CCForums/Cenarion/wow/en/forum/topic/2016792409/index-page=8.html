Please use the search feature that is within the site. This should hopefully help you find already existing threads on a topic of interest.
Hello!

I would like to suggest that the search function get some basic improvements.

1. Set the search bar to search within the current web section or forum page by default. Add a second button or radio button to instead search the entire website.

The largest problem is that a search, by default, searches the entire community site. This produces thousands of results, and is often useless. Often individuals know where best to find the information, and frequently, they have already navigated to that section. For instance, a few days ago I searched for "frost mage pvp spec"; it returned hundreds of results within the DK forum.

2. Provide an Advanced Search page.

While there are ways of narrowing the search by meta-information, they are very cumbersome. For instance, to narrow a search to only this forum (General Discussion), you must include forumId:984270 (or something similar). The syntax needs to be looked up and the ID codes also have to be found, often by visiting the forum or page and then copying from the address bar.

A Blizzard poster replied to this request by saying that these additions of syntax aren't that difficult, and so adding advanced options was not a priority. On the contrary, it is often the case that people will write a new thread with the same information before looking up the proper search syntax. I think Neth's request here (for us to search more often) may indicate that the search capabilities are cumbersome and/or lacking.

3. A great start to helping to categorize search results are the tabs for All, Arena Teams, Guilds, Forums, Characters, Items, etc. Good idea. I would like to see this expanded so that when Forums is clicked, a new set of tabs is opened up for each forum.

Thanks a bunch!

Here's an issue I feel needs to be addressed and it seems disturbingly insignificant here. Its almost like "well thats the way it is, if you don't like it, leave."

There are going to be disagreements about forum moderator actions, especially concerning the extremely vague explanation of "trolling" in the rules of conduct, and the moderator abuse of characterizing a post as trolling when its not. If a moderator disagrees with this post they could characterize it as trolling, remove it and ban me for any legnth of time if not permanently. The moderator could say it causes" forum unrest" as any uncomfortable subject would. My only recourse would be to appeal to the moderator via wowreportedpost@blizzard.com.

Thats akin to having a grievance with a building contractor, and the only recourse you have is appealing to the building contractor. You tell the contractor that he did something wrong, and he stonewalls you and says he didn't. There would be no arbitration, no fresh eyes, the contractor's word is law.

This forum has the same dynamic in play. A costumer posts something he considers innoccuous and is disciplined in a way he feels is unfair. He has absolutely no recourse but appeal to the same moderators who are not constrained from stonewalling him. So while you watch moderators exchange friendly banter about ridiculous musings ( Worgen nipples? Favorite poets? What movie star would play your main character?), you find yourself on the ouside looking in, and there is absolutely nothing you can do about it.

I understand "thats the way it is, if I don't like it, I can leave."

I don't like it, I did, but I still have 30 days on my account.

I'll get over it, but there is something very broken in the dynamic in play for addressing grievances on this forum.
