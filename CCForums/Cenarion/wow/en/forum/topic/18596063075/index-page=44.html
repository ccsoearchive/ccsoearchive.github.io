Why can i log into my server on some toons and get world server down with others on same server?

Previously answered by the present Blue
[quote="186913991652"]Anyone else having an issue where they can log onto some characters but not all? I can't log onto my priest on Blackrock-US but I can log onto some others.


Some locations, like your Garrison, exist on an Instance (the same realms that run Raids, Dungeons, and Battlegrounds that don't necessarily exist on the same realm as a World).

So, if the Instance Realm that you're on is good but the World Server you're on is experiencing issues, when you try to leave the transfer between the two will fail and keep you in place to prevent you from getting disconnected and then stuck on that bad zone.
