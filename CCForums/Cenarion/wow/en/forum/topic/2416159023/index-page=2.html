@Ragingkenny and Rapidomuerte:

I think there might be a misunderstanding on how this feature works. Minimizing or locking the screen will not log you out, instead it will put you into a Mobile Away state.

While in this away state, you will receive whisper messages as push notifications. Guild Chat and Officer Chat messages will NOT be received as push notifications, only Whispers from guild members.

You have a window of about 30 minutes to return to the Mobile Armory in order to see guild chat sent while you were away. So if you return within 30 minutes, you'll see guild chat history, but after that, the server will drop all of that history, except whispers. Whispers are not dropped from history and should be stored in their own individual message windows from particular guild members.
