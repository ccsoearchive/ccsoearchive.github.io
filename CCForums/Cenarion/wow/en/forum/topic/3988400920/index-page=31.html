It's a part of it. Much like LFD/LFR the current design teams seems to want the game to be more 'democratic' so to speak. The homogenization was part of that push..

Here is the obvious problem - with homogenization you boil the game down to 4 classes. That is the only logical outcome if you truly homogenize everything. So in every circumstance if a melee DPS has to do as well as another Melee DPS you end up with essentially the same class. This is especially true in PvP because even the smallest advantage will get magnified..

For example if Ferals can't do as much DPS as a rogue as "burst' they are underpowered in comparison - and this can be exploited. If OTOH Bears are very sturdy and hard to kill - they might have a signifcant advantage over rogues - and this will need to get nerfed. And this cycle continues until 'balance' is achieved and every class plays the same.

Personally I think this is coming from B. Kotick. Because I do not believe for a minute this game is the game that the developers want to play. Back in TBC and Vanilla it was very clear that WoW was the game that developers dreamed of playing for themselves.

Now its all about trying to please everyone. And that really hurts the game. I know the developers used to actively play and Blizzard had a bunch of players from its offices playing the game every single day. They need to go back to that plan so they can be less reactive and start making the game they want to play again..
