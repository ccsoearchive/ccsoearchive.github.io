A long time ago on a Warcraft server far far away, there was once an open Ventrilo server. A server where players from all over the realm could meet and converse and talk and roleplay and... well, whatever really!

And so, my fellow Accordians, I present to you, your very own Ventrilo Server.

Kerielle generously hosted this server originally, and now it has been passed into my care. I have renewed it for another year, and will do my best to keep it a safe and friendly place.

Should you ever have any issues with this Ventrilo server (trolls, bans, technical issues, ect) please feel free to leave a message for me here, or contact me in game. I will respond and address your issue as soon as possible. Otherwise I'm usually sitting in vent for a few hours each day, if you want to reach me there.

-------------------------------------------
Name: Wyrmrest Accord
Host: oxygen.typefrag.com
Port: 7939

The lobby is silent, make sure you change channels to speak.

The server can currently handle 35 people. If the demand is there, I'm happy to increase it.

Trolls, harassment, or any other bad behavior will NOT be tolerated in any way on this Ventrilo server. So use it wisely!

-------------------------------------------
New-to-Vent set up instructions:

Make sure you have the right sound device selected.
Try this:
~ Log in to vent.
~ Click the setup button.
~ On the left-hand-side make sure you have Enable outgoing voice communications ticked.
~ Make sure you are using the Speex codec at 32/16/10
~ On the right-hand-side make sure you tick Use Direct Sound for both the input and output device.
~ Make sure the device you have picked for both input and output is your headset.
(mine says "Speakers (Logitech USB Headset)".)
~ Click OK.
~ Make sure at bottom of the main screen you haven't accidentally clicked Mute Sound.
~ Make sure you haven't accidentally muted your mic from the headset itself.

In order to keep this Ventrilo server a friendly and safe place for everyone, there is a zero-tolerance policy towards harassment or trolling.

If I or any of my fellow administrators catch any bad behavior, we will ban the offender, no questions asked.

If you are having problems with someone on vent either open a private chat with anyone in the Admin channel (right-click their name > User > Private chat) or you can leave a complaint notice (right-click the offender's name > User > Lodge complaint about).

You can also contact myself (Adellwyna) or my other admins (Lierona, Alesune, Xeqwena, Natasta or Aeraniel) in-game via whispers or mail and we'll address your issue as soon as possible.

Finally you may email us at sentinelwra@gmail.com.
