Thanks to the people who have sent logs in so far. It looks like these have to do with permissions problems for the Mac logs I've seen so far. Let's make sure that your game files and the Battle.net App have necessary permissions:

    1. Using Finder, navigate to the /Applications directory.
    2. Right-click the game folder and select Get Info.
    3. In the info box, go to the Sharing & Permissions section and click on the lock to unlock it. Enter your administrator password when prompted.
    4. Make sure all privileges are set to Read & Write.
    5. Click on the Action icon (cogwheel) and select Apply to all enclosed items. Select Yes to confirm the action.


Note: Step 5 must be done, even if the folder already displays Read & Write privileges. This helps ensure all enclosed files have the same permissions.

Additionally, try [url="https://us.battle.net/support/article/6572?utm_source=Internal-TechForum&utm_medium=Posting&utm_campaign=BlizzardCS&utm_content=solution"]running a Scan & Repair[/url]. This should verify any permissions files and try to elevate them as necessary.

Edit: Gartan, the Scan & Repair should work for you if the issue is the same. If not, post over in our Technical Support Forums so our PC Tech Team can investigate!
