I have a solution.

1. ) Make a new macro (type /macro, hit enter), name it anything you like, choose any icon, does not matter.

/run for 1,(CalendarGetNumPendingInvites()) do for i=0,90 do for y=1,31 do t,_,_,_,_,_,_,_,_,iB =CalendarGetDayEvent(i,y,1) if iB=="" or t=="TITLE" then CalendarContextInviteRemove(i,y,1) end end end end

2. ) Copy+Paste the above code into the macro window

Replace TITLE, with the name of all these events (they will most likely ALL be the same)

3.) Click the macro.

:), I hope this helps
